# Khuda-8th_YoutubeThumbnail-AI
Khuda 8th Databusiness Track. YouTube Thumbnail Analysis Project Using Explainable AI (xAI)

## 1. 프로젝트 설명
본 프로젝트는 썸네일의 시각적 정보와 감정 강도를 멀티모달 AI로 분석하여 썸네일이 조회수에 미치는 순수 효과를 예측하고, Grad-CAM 기반의 시선 추적 히트맵을 통해 설명 가능한 인사이트(XAI)를 제공합니다. 
나아가  히트맵을 기반으로 생성형 AI(Gemini)를 활용해 사용자에게 피드백을 제시하는 프로젝트입니다.
